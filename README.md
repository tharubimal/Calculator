Calculator

A fast and modern scientific calculator web app built with HTML, CSS, and JavaScript.

🚀 Features
Basic operations (+, −, ×, ÷, %)
Scientific functions (sin, cos, tan, log, ln, √)
Power calculations (x², xʸ)
Constants (π, e)
Live result preview
Keyboard support
Calculation history
Dark / Light theme toggle
🛠️ Tech Stack
HTML, CSS, JavaScript
Custom expression parser (no eval())
▶️ How to Use
Enter numbers using buttons or keyboard
Choose operations or functions
Press = to calculate

Open index.html in your browser — no setup needed.

📌 Note

This calculator uses a custom-built parser with the Shunting Yard algorithm for safe and accurate evaluation.
